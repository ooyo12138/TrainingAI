# 训练任务索引

这套任务专门练习 Codex 的高级协作能力。

## 顺序

1. `01-local-page-baseline.md`：启动本地页面，理解虚拟项目。
2. `02-worktree-lab.md`：用 Worktree 隔离一个功能改动。
3. `03-automation-lab.md`：设计并测试一个 Codex Automation 提示词。
4. `04-subagent-lab.md`：显式让 Codex 派多个 subagent 并行审查。
5. `05-custom-mcp-page.md`：给本地页面启用自定义 MCP。
6. `06-integrated-challenge.md`：把 Worktree、Automation、Subagent、MCP 串起来。

## 推荐开始提示词

```text
请先不要修改文件。
请阅读 tasks/01-local-page-baseline.md，并带我完成第一个练习。
```
