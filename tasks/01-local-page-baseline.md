# 任务 01：本地页面基线

## 目标

先确认训练项目能跑，理解页面、数据和测试。

## 背景

`virtual-project/` 是一个普通的任务看板页面，数据来自本地 JSON 文件。

## 要求

让 Codex 阅读：

- `virtual-project/README.md`
- `virtual-project/AGENTS.md`
- `virtual-project/server.js`
- `virtual-project/src/`
- `virtual-project/public/`
- `virtual-project/tests/run-tests.js`

## 验收标准

Codex 应该告诉你：

1. 页面做什么；
2. 数据存在什么文件；
3. 服务怎么启动；
4. 测试怎么运行；
5. 后续练习会用到哪些文件。

## 推荐提示词

```text
请先不要修改文件。
请根据 tasks/01-local-page-baseline.md 阅读 virtual-project。
```
